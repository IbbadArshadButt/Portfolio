
## Overview
This is a professional portfolio website built using **HTML**, **CSS**, and **JavaScript**. The website showcases personal projects, skills, and includes a contact form. The contact form is integrated with **Formspree** to handle form submissions without the need for a backend.

## Features
- Responsive design optimized for different screen sizes.
- A clean and modern layout with sections for:
  - **Home**: A brief introduction.
  - **About**: Details about skills, education, and experience.
  - **Projects**: Showcasing some of the most important projects.
  - **Contact**: A form that allows users to send messages directly to your email using **Formspree**.


## Technologies Used
- **HTML5**: Structure of the website.
- **CSS3**: Styling the website to ensure a professional, elegant look and feel.
- **JavaScript**: Interactive elements such as form validation and dynamic content updates.
- **Formspree**: Used for sending form submissions to your email without a backend.
